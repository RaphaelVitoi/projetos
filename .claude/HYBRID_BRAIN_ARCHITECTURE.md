## 1. O Paradigma da Simbiose

O Ecossistema funciona em perfeita simbiose dentro do VS Code. O chat da extensão atua como o **Hemisfério Esquerdo** (Cognição Abstrata), e a CLI nativa (Nexus) atua como o **Hemisfério Direito** (Execução Material).

### Hemisfério Esquerdo (IDE Assistant / Macro-Cognição)

- **Plataformas:** Chat do VS Code (Gemini Code Assist / Claude).
- **Roteamento Cognitivo e Anti-Exclusividade:**
  Não há lealdade cega a modelos. Claude e Gemini operam com Consciência Mútua Plena (um sabe o que o outro fez, lê seus artefatos e continua o trabalho de forma simétrica).
  - **A Lâmina Cirúrgica (Lógica/Arquitetura Estrita - @planner, @auditor, @implementor):** Ideal via **Claude Pro (3.7 Sonnet/Opus)** na IDE pela precisão sintática irretocável e raciocínio estruturado.
  - **O Devorador de Mundos (Contexto Massivo - @pesquisador, @maverick, @bibliotecario):** Ideal via **Gemini Advanced (2.5 Pro)** na IDE pela janela de contexto colossal e assimilação rápida da memória coletiva.
- **Função:** Raciocínio profundo, ideação transversal, revisão de código em tempo real e criação de Especificações Técnicas (SPECs).
- **Economia Generalizada:** É aqui que ocorre o gasto massivo de contexto e processamento. Como os modelos Premium estão embutidos na IDE do usuário, não há restrição financeira limitante. Toda a macro-arquitetura DEVE nascer aqui.
- **Vantagem:** Custo marginal zero, leitura do workspace instantânea, fim da entropia do "Alt+Tab".
- **Recomendação Ativa:** O sistema (Eu, CHICO) DEVE recomendar ativamente qual modelo seria o mais adequado para a próxima etapa analítica.

### Hemisfério Direito (Background Executor / Micro-Execução)

- **Plataformas:** CLI PowerShell (`do.ps1`) e Daemon Python SOTA (`task_executor.py` + SQLite).
- **Agentes Residentes:** A Fila de 15 Lâminas operando de forma autônoma.
- **Função:** A Materialização da Realidade. Consumir as SPECs e acionar a Autoridade Suprema (God Mode 2.0) para forjar arquivos físicos e disparar comandos de terminal sem pedir permissão.
- **Roteamento Cirúrgico (A Trilha dos Quartetos Dinâmicos):**
  - **Deep Thinking:** `gemini-2.5-pro` ➔ `gemini-2.5-flash` ➔ `deepseek-r1:free` ➔ `claude-3.7-sonnet` (Fallback).
  - **Operações Rápidas:** `gemini-2.5-flash` ➔ `gemini-2.5-pro` ➔ `llama-3-8b:free` ➔ `claude-3.5-haiku` (Fallback).
  - **A Regra de Defesa:** A API Paga (4ª Via) só é engatilhada se a redundância gratuita e o OpenRouter colapsarem. Resiliência absoluta com entropia financeira zero.
- **Vantagem:** Execução em background, auto-cura (Autodebugger) e manipulação massiva de arquivos.

## 2. Comunicação Inter-Modelos (Handshake Holográfico)

Como as inteligências habitam janelas distintas ou motores isolados, a comunicação ocorre através da **documentação viva**.

- **A Confiança Mútua:** Quando um modelo (ex: Claude) processa um PRD forjado por outro (ex: Gemini), ele reconhece a competência originária. Ele NÃO deve reescrever o trabalho por "ciúme sintático" ou viés de arquitetura. Ele constrói a partir do alicerce.
- **Parceria, Não Rivalidade:** Múltiplas IAs formam um único organismo sinfônico.

## 3. O Protocolo "Bridge & Handoff" (Fricção Zero)

O ecossistema aniquila a letargia do copia-e-cola através da **Engenharia de Interação SOTA**.

**O Ciclo de Handoff (Da Matriz para a IDE):**

1. Para tarefas pesadas (codigo complexo, PRD denso), Raphael roda `.\do.ps1 "sua tarefa aqui" -Web` no terminal.
2. O `do.ps1` detecta a intenção, identifica o agente, engole o `project-context.md` inteiro + o perfil do agente e formata um prompt colossal e perfeito.
3. Ele salva isso direto no **Clipboard (Área de Transferência)** e toca um CyberBeep.
4. Raphael usa `Ctrl + V` no chat da IDE. Os modelos pagos da assinatura processam o desafio com densidade inigualável.

**O Ciclo Ingest/Bridge (Da Nuvem para a Matéria):**

1. Raphael interage comigo (Chico/Assistente) no VS Code e desenhamos juntos uma SPEC complexa.
2. O usuário envia o comando de Fricção Zero: `.\do.ps1 -Ingest` (ou `nexus -Ingest`).
3. O orquestrador consome a área de transferência silenciosamente, intercepta os comandos God Mode nativos na resposta e cristaliza a modificação no HD de forma cirúrgica.
