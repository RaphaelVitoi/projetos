### Executando Autocorreção e Backup

Primeiro, vou gerar um timestamp dinâmico para o nome do diretório de backup, seguindo o formato `YYYY-MM-DD_HHmmss` para consistência.

```powershell
$timestamp = Get-Date -Format "yyyy-MM-dd_HHmmss"
$backupDirectoryName = "full_backup_$timestamp"
$backupPath = "backups/$backupDirectoryName"

Write-Host "Criando diretório de backup: $backupPath"

# Comando para criar o diretório de backup (corrigido para PowerShell)
New-Item -ItemType Directory -Path "$backupPath" -Force

Write-Host "Diretório de backup criado com sucesso."
Write-Host "Iniciando cópia dos arquivos do projeto para $backupPath..."

# Comando para copiar os arquivos do projeto para o diretório de backup
# .           -> Origem (diretório atual, assumindo que é a raiz do projeto)
# $backupPath -> Destino (o diretório de backup recém-criado)
# /E          -> Copia subdiretórios, incluindo vazios
# /COPYALL    -> Copia todas as informações de arquivo (atributos, carimbos de data/hora, segurança, proprietário, auditoria)
# /DCOPY:T    -> Copia os carimbos de data/hora dos diretórios
# /XF "*.log" -> Exclui todos os arquivos com extensão .log (geralmente logs temporários)
# /XD "backups" -> Exclui o próprio diretório 'backups' para evitar cópia recursiva
# /NFL        -> Não lista os nomes dos arquivos
# /NDL        -> Não lista os nomes dos diretórios
robocopy . "$backupPath" /E /COPYALL /DCOPY:T /XF "*.log" /XD "backups" /NFL /NDL

Write-Host "Backup de 'tudo que fizemos hoje' concluído com sucesso em $backupPath."
```
