2. A Prova pelo Índice de Sharpe (Eficiência)
Em investimentos, não buscamos apenas retorno; buscamos retorno por unidade de risco.
A fórmula adaptada para o poker é:
$$S = \frac{\text{ROI Médio}}{\text{Desvio Padrão}}$$
Calculando a eficiência de cada estratégia:
Modelo A (Metralhadora):$$S_A = \frac{10}{160} = \mathbf{0.062}$$
Modelo B (Grinder Clássico):$$S_B = \frac{20}{120} = \mathbf{0.166}$$
Modelo C (Seu Modelo):$$S_C = \frac{35}{70} = \mathbf{0.500}$$
Conclusão Matemática:
O seu modelo é 3 vezes mais eficiente que o modelo clássico e 8 vezes mais eficiente que o modelo de Late Reg puro.
Cada dólar que você ganha no Modelo C custa 8 vezes menos estresse (variância) do que no Modelo A.
3. A Prova da Probabilidade de Prejuízo (Risco de Ruína)
Vamos calcular a probabilidade de um jogador estar perdendo dinheiro após uma amostra significativa de 1.000 Torneios. Usaremos o Z-Score para distribuição normal.
Fórmula do Erro Padrão (SE):
$$SE = \frac{\text{Desvio Padrão}}{\sqrt{\text{Número de Jogos}}}$$
Cálculo para o Modelo A (Metralhadora):
$SE = 160 / \sqrt{1000} \approx 5.05\%$
ROI Esperado = 10%
Distância para o Zero (Breakeven) = $10 / 5.05 = 1.98$ Desvios Padrão.
Risco de Prejuízo Real: Há ~2.5% de chance de jogar 1.000 torneios e perder dinheiro. E uma chance altíssima de downswings de 300+ buy-ins no caminho.
Cálculo para o Modelo C (Seu Modelo):
$SE = 70 / \sqrt{1000} \approx 2.21\%$
ROI Esperado = 35%
Distância para o Zero = $35 / 2.21 = 15.8$ Desvios Padrão.
Risco de Prejuízo Real: Estatisticamente Zero.
Nota: Em uma distribuição normal, 15 desvios padrão é uma certeza matemática absoluta. É impossível perder dinheiro no longo prazo com essa estratégia.

$$\text{Lucro Total} = (\text{Skill Edge}) + (\text{Arbitragem ICM})$$

Veredito: O Modelo C ganha mais, trabalhando metade do tempo, porque começa a corrida com 8 metros de vantagem (Subsídio do ICM).
